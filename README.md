# Sistema IdLab
É um sistema completo para o projeto de extensão de Design thinking, contendo painel de administração, postagens e criação de contas.

# Instalação 

``
npm install
``

``
php artisan install
``

``
php artisan key:generate
``

``
npm run dev
``

``
cp .env.example .env
``

# Incialização 

``
php artisan serve
``
